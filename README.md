# FX News Dashboard

Full-stack macroeconomic dashboard using:

- React + Vite
- Flask
- Socket.IO
- TradingView Widgets
- JWT Authentication
- SQLite Database
- Render Deployment

## Features

- CPI/NFP dashboard
- Auto-refresh
- WebSocket live updates
- DXY chart
- Gold chart
- Event filtering
- Dark/light theme
- Mobile responsive sidebar
- Authentication
- Signal history storage
